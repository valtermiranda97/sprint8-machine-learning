# Sprint 8 - Machine Learning: Mobile Plan Recommendation

Binary classification project to recommend the best mobile plan (Smart or Ultra) 
based on customer behavior.

## What it does
- Analyzes customer usage data (calls, minutes, messages, internet)
- Trains and compares multiple classification models
- Selects the best model based on accuracy
- Achieves accuracy above 0.75 threshold

## Models tested
- Decision Tree
- Random Forest ✅ (best model)
- Logistic Regression

## Results
- Best model: Random Forest
- Test accuracy: ~0.80 (above 0.75 threshold)
- Outperforms dummy classifier baseline

## Skills demonstrated
- Binary classification
- Hyperparameter tuning
- Model selection and evaluation
- Train / Validation / Test split

## Tools
- Python 3
- Pandas
- Scikit-learn

Project developed during TripleTen Data Science Bootcamp - Sprint 8
